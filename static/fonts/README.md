# Pokémon Font

To use a Pokémon-themed font for headings:

1. Download "Pokemon Classic" from: https://www.dafont.com/pokemon-classic.font
   - Or "Pokemon GB" from: https://www.dafont.com/pokemon-gb.font

2. Place the `.ttf` file in this directory (`static/fonts/`)

3. Rename it to `Pokemon-Classic.ttf`

4. The font will automatically be loaded and applied to all `<h1>` headings

**Note:** These fonts are typically free for personal use. Check the license for commercial use.
